# Floating Subtitles
F4SE plugin that adds floating subtitles over NPCs

## OG (1.10.163) Support

The plugin supports the pre-Next-Gen version of Fallout 4 (1.10.163). Most features work identically to the NG version, with the following limitations:

- **No dynamic color updates** — HUD and subtitle colors are read from INI at startup. Changes to HUD color in settings require a game restart to take effect.
- **No dual-language subtitles** — Only the game's native language is displayed. The secondary language overlay is not available.
- **No crosshair highlighting** — Subtitles don't change appearance when looking directly at an NPC.
